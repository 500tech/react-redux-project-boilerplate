## Requirements

Git version must be `>= 2.13.2`
Node version must be `>= 8`

## How to use

```
$ git clone git@github.com:500tech/react-redux-project-boilerplate.git
$ git remote set-url origin <the-specific-repo-git-url>
$ yarn start
```
